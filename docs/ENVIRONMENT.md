## Environment variables

Required:
- `NEXHEALTH_API_KEY` NexHealth API key
- `NEXHEALTH_SUBDOMAIN` practice subdomain
- `NEXHEALTH_LOCATION_ID` location id

Optional:
- `NEXHEALTH_BASE_URL` default `https://nexhealth.info`
- `NEXHEALTH_API_VERSION` default `v2`
- `NEXHEALTH_ACCEPT` default `application/vnd.Nexhealth+json;version=2`
- `APP_ENV` default `local`
- `LOG_LEVEL` default `info`
- `ADMIN_API_KEY` default `secret-admin-key` (Change in prod!)
- `RETELL_API_SECRET` (Required for voice agent)
