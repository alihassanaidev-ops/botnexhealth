## Drafting Assumptions

- Initial market is Canadian dental clinics.
- Ontario is included in the initial Canadian scope.
- Patients submit access/correction/privacy requests to their dental clinic
  first. Clinics may escalate platform-specific requests to Scale Nexus.
- Founder is the current privacy/security accountable role.
- Production vendors are AWS, Retell, Twilio, Resend, and NexHealth.
- Call recordings default to 90-day retention.
- Resend sends auth emails, staff call notifications, and a gated
  patient-facing appointment confirmation if a clinic explicitly enables that
  template.
